# Lab7 — Односпрямований список (float)

Лабораторна робота №7 з дисципліни «Основи програмування. Частина 2».
Тема: побудова та використання структур даних.

## Варіант завдання

- Тип елементів списку: `float`
- Тип списку: односпрямований
- Спосіб додавання елементу: вставка після другого елемента списку

## Реалізовані операції

1. Знайти перший від'ємний елемент списку.
2. Знайти суму елементів списку, більших за середнє значення.
3. Отримати новий список зі значень позитивних елементів поточного списку.
4. Видалити всі від'ємні елементи поточного списку.

Додатково реалізовано:
- доступ до елемента списку за індексом (`list[i]`);
- перебір елементів списку через `foreach` (`IEnumerable<float>`);
- валідація консольного вводу (повторний запит при некоректному значенні);
- коректна обробка порожнього списку.

## Структура проєкту

```

├── Node.cs               # вузол однозв'язного списку

└── Lab7.csproj
```

## Запуск

Потрібен встановлений .NET SDK 9.0.

```bash
dotnet run
```

Програма запитає кількість елементів, а потім значення кожного елемента.
Перші два елементи додаються в кінець списку, кожен наступний — одразу
після другого елемента.

## Приклад роботи

```
Введіть кількість елементів: 5
Введіть елемент 1: 1
Введіть елемент 2: 2
Введіть елемент 3: 3
Введіть елемент 4: -4
Введіть елемент 5: 5

Середнє арифметичне елементів списку: 1.4
Перше знайдене від'ємне значення: -4
Сума значень, які перевищують середнє: 8
Список, сформований лише з позитивних елементів:
1 2 3 5
Список після вилучення всіх від'ємних значень:
1 2 5 3

Перебір списку через foreach:
1 2 5 3
```
